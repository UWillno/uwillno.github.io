# uwillno.github.io

## 介绍

基于Qt for WebAssembly开发的个人博客系统，内置一些自用功能。

## 注意

- 已经放弃追求稳定，尽量采用最新Qt版本构建，有的版本稳定性很差，遇见无响应或者崩溃我只建议刷新。
- 目前我测试过很稳定、长时间使用过的版本为**Qt 6.7.2**。虽然WASM上有些模块有问题，但是可以靠编码来回避。如果只实现博客功能，基础的模块是完全够用的。
- Qt版本更新可能会引入新功能，解决旧问题，但可能**也会导致靠编码来回避的方法失效或白写了、带来其它更不稳定的问题（~~靠编码也回避不了，可能得改Qt的源码~~）**，**不建议针对WebAssembly平台折腾。**
- **.wasm文件比较大，注意流量消耗。**
